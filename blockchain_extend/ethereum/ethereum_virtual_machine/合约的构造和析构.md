# 合约的构造和析构

## 只有构造函数的合约
construct
```
pragma solidity 0.5.16;
contract Test1 {
    uint a;
    uint b;
    constructor(uint _a) public {
        a = _a;
        b = 200;
    }
}
```

opcode
```
0: PUSH1 0x80
1: PUSH1 0x40
2: MSTORE
3: PUSH1 0x00
4: DUP1
5: REVERT
```
